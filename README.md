# Elden Ring Completion Companion

A single-page web app that helps Elden Ring players reach 100% item collection by analyzing their save file and guiding them toward missing items.

## Planned Features

- Automatically refresh when the selected save file changes
- Detect collected and missing items from the player's save file
- Support multiple character slots
- Display completion progress by item category
- Provide an overview of all missing items
- Provide a guided mode that shows one missing item at a time with acquisition instructions
- Run entirely in the browser using the File System Access API
